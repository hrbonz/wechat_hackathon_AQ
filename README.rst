#######################################
Masks - masks for all (wechat mini-app)
#######################################

Provide valuable information about masks and pollution to people.


Development
===========

Backend
-------

.. code:: sh

    $ mkvirtualenv --system-site-package maskapp
    $ pip install -r requirements-dev.txt

Run test API

.. code:: sh

    $ cd backend
    $ python rundevapi.py
